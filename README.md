# dogfriendlybeach

Dog-friendly UK beach finder (GitHub Pages app).

## Features
- address search (town/city/postcode)
- radius filtering
- map + directions links
- dog-access notes (year-round/seasonal)
- beach-themed UI

## Data
This first version uses a curated starter dataset of UK beaches.
Always check local signage and council guidance before visiting.

## Deploy
Automatic deploy to GitHub Pages using GitHub Actions in `.github/workflows/deploy-pages.yml`.
